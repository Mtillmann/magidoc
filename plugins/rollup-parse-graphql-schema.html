Rollup Parse GraphQL Schema

Standalone
Plugin
Rollup
Vite
Since: 1.0.0

Similar to what the Rollup Fetch Plugin does, the Rollup Parse Plugin generates an introspection schema from your SDL files and stores the result on the file-system. This approach is usually simpler than interacting with a live endpoint if your API uses Authentication.

This plugin is also compatible with ViteJS , which is used by SvelteKit.

Install

First, add the module to your project.

Then, add the plugin to your Rollup/ViteJS configuration. Here is an example for a svelte.config.js.
